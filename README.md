## 2D Кубі-бубі

## Описання:

2D Кубі-бубі - це HTML-головоломка, натхненна класичною грою "Кубики Рубіка". Гравці обертають ряди та стовпці, щоб переміщати блоки та збирати їх у відповідні лінії.

## Особливості:

    Простий і зрозумілий ігровий процес
    Різні рівні складності
    Чиста та мінімалістична графіка
    Локально працює в браузерах мобільних телефонів

## Як грати:

    Виберіть рівень складності.
    Обертайте ряди та стовпці, щоб переміщати кольорові блоки.
    Зберіть всі блоки в відповідні комірки.

## Поради:

    Починайте з низького рівня складності, щоб ознайомитися з ігровим процесом.
    Скористайтесь кнопкою "автогра" щоб зрозуміти механіку складання
    Ви можете використовувати кнопку "Скинути", щоб почати рівень заново.
